ClusterV-Web

ClusterV-Web is a web application for accurately identifying HIV subtypes from ONT sequencing data.

ClusterV-Web takes the alignment BAM file, reference FastA file and target region BED file (defining the target regions) of the HIV data as input, and outputs all found subtypes with their abundance, alignment, variants (SNPs and INDELs), and the drug resistance reports.
